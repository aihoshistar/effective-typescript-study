### Item 4. 구조적 타이핑에 익숙해지기

- 자바스크립트는 기본적으로 덕 타이핑이다.

### Item 5. any 타입 지양하기

- 타입스크립트의 타입 시스템은 점진적(Gradual), 선택적(Optioanl)이다.
- 타입스크립트에서 Any 를 쓰지 말아야 하는 이유가 많고, 써야 하는 이유는 적다.
  - Any 는 함수 컨트렉트를 무시한다.
  - 리펙토링에 방해되고 버그를 찾는데 방해된다.
  - 타입 설계를 감춘다.
  - 타입시스템의 신뢰도를 떨어트린다.
